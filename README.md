
🎵 Melody Store Logs

📝 About

An e-commerce platform for premium digital accounts and services. Customers can buy verified social media accounts, VPN subscriptions, and digital services with instant delivery after payment.

✨ Features

· Paystack payment integration
· Automated email delivery of credentials
· Admin dashboard with full management
· Responsive design
· Order tracking system

🛠️ Tech Stack

· Flask (Python)
· PostgreSQL
· Paystack API
· HTML/CSS/JS

📦 Products

· TikTok, Instagram, Twitter, Facebook accounts
· VPN subscriptions
· Texting apps
· iCloud & Apple Music codes

🚀 Quick Start

# Clone
git clone https://github.com/kariosv4sure/Melody-Store---Premium-Accounts-Marketplace-.git

# Install
pip install -r requirements.txt

# Set environment variables 

# Run
python app.py

Visit: http://localhost:5000

Admin: /admin/login

📧 Email Setup

Use Gmail App Password for SMTP.

🚢 Deploy

Deploy on Render with PostgreSQL.
